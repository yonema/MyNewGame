●操作説明
・Aボタン		：ジャンプ

・RT（R2ボタン）	：（地上で）ダッシュ、（空中で）スイング
・左スティック		：移動
・右スティック		：カメラ回転
・右スティック→押し込み：敵探知
・STARTボタン		：ミッション確認
